Exercícios com Vega-Lite API

Exercício 1 (barras agregadas) e Exercício 2 (dois scatterplots lado a lado) integrados via Observable.

Gráficos gerados no Observable e embutidos via iframe (células bar e scatters).

O que o Exercício 1 mostra?

Barra agregada: média de Avg_BPM por Workout_Type, ordenada em ordem decrescente. O objetivo é comparar a intensidade cardiovascular média entre modalidades. HIIT tende a apresentar os maiores valores médios.

O que o Exercício 2 mostra?

Dois scatterplots, um para Homens e outro para Mulheres, relacionando Peso (kg) (X) e Altura (m) (Y). Cada ponto é um indivíduo. As escalas X e Y são compartilhadas para comparação justa. Observa-se a tendência positiva esperada: pessoas mais altas tendem a pesar mais.
